Ensures the array has capacity for at least to_reserve elements.

If the current capacity already meets or exceeds to_reserve, this function does nothing. Otherwise the internal buffer is reallocated.

Parameters
arrThe array. Must not be NULL.
to_reserveMinimum capacity to guarantee. Must not exceed the value returned by array_max_size.
errOptional error output. Populated on failure.
