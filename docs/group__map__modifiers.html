Inserts a key-value pair, or updates the value if the key exists.

If the load factor exceeds max_load_factor after insertion, the table is automatically resized and all entries are rehashed.

Parameters
mapThe hash map. Must not be NULL.
keyPointer to the key data. Must not be NULL.
valuePointer to the value data. Must not be NULL.
errOptional error output. Populated on failure.
