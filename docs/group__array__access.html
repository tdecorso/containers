void * array_at ( array_t *  arr,
size_t  index,
error_t *  err 
)

Returns a pointer to the element at the given index.

Performs bounds checking against cap (not count), so sparsely inserted elements beyond count are reachable.

Parameters
arrThe array. Must not be NULL.
indexIndex of the element to access. Must be < cap.
errOptional error output. Populated on failure.
Returns
Pointer to the element, or NULL if the array is empty and index is 0, or if index is out of bounds.
