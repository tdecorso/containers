Returns the end iterator for the list.

The end iterator represents a position past the last element and is used as a termination condition during iteration.

Parameters
listThe list. Must be not NULL.
errOptional error output information. If non-NULL and the call fails, it will contain details about the failure.
Returns
Always returns NULL (end sentinel). Provided for API symmetry and future extensibility.
Note
This function does not modify the list.
