list_node_t *  node,
void *  item_out,
error_t *  err 
)

Removes a node from the list.

The node is detached from the list and its memory is freed.

Parameters
listThe list. Must be not NULL.
nodeNode to remove. Must be a valid node belonging to list.
item_outOptional output buffer. If non-NULL, the removed element's data is copied into it before the node is freed. Must be large enough to hold elem_size bytes.
errOptional error output information. If non-NULL and the call fails, it will contain details about the failure.
Note
Complexity: O(1)
After this call, node becomes invalid and must not be used.
Invalid node (not belonging to list) results in undefined behavior.

Inserts a new element after the specified position.

The new node is inserted immediately after pos.

Parameters
listThe list. Must be not NULL.
posIterator (node) indicating the insertion position. Must be a valid node in the list or NULL. If NULL, the element is inserted at the front of the list.
itemPointer to the data to insert. Must be not NULL. The data is copied using elem_size.
errOptional error output information. If non-NULL and the call fails, it will contain details about the failure.
Note
Complexity: O(1) if inserting at known node.
If pos is NULL, insertion happens at the front of the list.
Invalid pos (not belonging to list) results in undefined behavior.
