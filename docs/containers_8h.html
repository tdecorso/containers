enum error_e

Error codes returned by the API.

These values indicate failure conditions and define how the caller should react.

Enumerator
ERROR_OK 

No errors detected.

ERROR_INVALID_ARGS 

One or more arguments are invalid; caller must fix input before retrying.

ERROR_OUT_OF_MEMORY 

Memory allocation failed; operation cannot proceed, retry may succeed if memory is freed.

ERROR_INVALID_STATE 

Internal state is corrupted; operation cannot proceed.
