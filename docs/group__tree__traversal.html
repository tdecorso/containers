tree_walk_fn  fn,

Visits every node in post-order (children before parent).

The callback may return TREE_WALK_STOP to abort traversal. TREE_WALK_SKIP_CHILDREN has no effect in post-order since children are visited before the callback fires; it is treated as TREE_WALK_CONTINUE.

Parameters
treeThe tree. Must not be NULL.
fnCallback invoked for each node. Must not be NULL.
userdataOpaque pointer forwarded to every callback invocation.
errOptional error output. Populated on failure.
Note
Complexity: O(n) where n is the total number of nodes.

Visits every node in pre-order (parent before children).

The callback may return TREE_WALK_SKIP_CHILDREN to skip the descendants of the current node, or TREE_WALK_STOP to abort traversal entirely.

Parameters
treeThe tree. Must not be NULL.
fnCallback invoked for each node. Must not be NULL.
userdataOpaque pointer forwarded to every callback invocation.
errOptional error output. Populated on failure.
Note
Complexity: O(n) where n is the total number of nodes.
